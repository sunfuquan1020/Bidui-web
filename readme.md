## 用法说明（如何替换占位图）
1. 准备好对应实拍图，按上表**文件名**命名（如 `portable-analyzer.jpg`）。
2. 放入项目的 `images/` 目录。
3. 修改 `index.html` 中对应 `<img>` 标签的 `src` 扩展名：
   - 产品卡：`<img src="images/portable-analyzer.svg" ...>` → 改成你的扩展名（如 `.jpg`）。
   - 应用卡同理（`app-datacenter.svg` 等）。
   - ⚠️ 直接用栅格图覆盖同名 `.svg` 文件不可靠（浏览器按内容而非扩展名渲染），务必改 `src` 扩展名。
4. 中英文文案已在 `index.html` 底部 `translations` 对象中维护（`zh` / `en`），切换按钮位于页面右上角；新增/修改产品或应用卡时，需同步 `products.cardN` / `applications.cardN` / `others.itemN` 三处键，并确保 `.product-card` 数量与 `cardN` 数量一致（语言切换 JS 按索引映射）。
